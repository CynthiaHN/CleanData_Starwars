# Basic data cleaning using StarWars dataset
In this project, the 'Starwars' data from dplyr in R was used to demonstrate how to clean data.

## Aim: 
+ Explore and identify data
+ Clean data 

## Task Performed:
### R Studio 
+ glimpse()
+ rename()
+ class()
+ unique()
+ levels()
+ select() filter()
+ na.omit()
+ drop_na()
+ mutate()
+ recode()
